TTS Everything

For TTS Everything to start working, you will need to enable the feature on the TTS Dashboard. From here you can also manage the voice used and wether or not the chatter's name should be read before each message. The selected voice will apply to both TTS Everything and the TTS Chat Command.

After this, any message sent will be read out in the selected voice

info

Messages read count towards your Monthly Quota, and it is not possible to use AI voices with TTS Everything.
